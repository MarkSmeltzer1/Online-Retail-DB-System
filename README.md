# Online Retail Database System

## Overview

Designed and implemented a relational database system for an online retail platform, supporting product management, customer transactions, and order processing.

---

## Features

* Product catalog management
* Customer accounts and orders
* Inventory tracking
* Payment and shipping information

---

## Database Design

* Designed normalized relational schema (up to 3NF)
* Defined relationships between entities (customers, orders, products)
* Implemented primary and foreign key constraints

---

## Technologies Used

* MySQL
* SQL

---

## Example Queries

* Retrieve top-selling products
* Analyze customer purchasing behavior
* Track order history and revenue

---

## Key Learnings

* Database normalization and schema design
* Writing efficient SQL queries
* Structuring data for real-world applications

---

## Future Improvements
